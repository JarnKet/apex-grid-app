# Widget Gallery Guide

## Overview

The Widget Gallery is an app drawer-style interface for browsing and adding widgets to your ApexGrid dashboard. It provides a visual, intuitive way to discover and add widgets with previews and detailed information.

## How to Access

Click the **"+"** button in the top-right corner of your dashboard (next to the Settings gear icon).

## Features

### 1. Category Sidebar
Browse widgets by category:
- **All Widgets** - View all available widgets
- **Productivity** - Clock, Calendar, Todo List, Quick Links
- **Information** - Weather, Daily Quote, Tech News
- **Finance** - Crypto & Gold prices

### 2. Widget Grid
- Visual cards showing each widget with:
  - Icon representing the widget type
  - Widget name and description
  - "Add Widget" button
  - "Added" badge if already on your dashboard
- Click any card to see more details in the preview panel

### 3. Preview Panel
When you click a widget card, the preview panel shows:
- Large icon and widget name
- Category tag
- Detailed description
- Feature list
- "Add to Dashboard" button
- "Close Preview" button

## Available Widgets

### Productivity
- **Clock** - Real-time clock with seconds and full date
- **Calendar** - Interactive monthly calendar
- **Todo List** - Task management with persistent storage
- **Quick Links** - Customizable website shortcuts

### Information
- **Weather** - Current conditions with temperature, wind, and humidity
- **Daily Quote** - Inspirational quotes with author attribution
- **Tech News** - Latest tech news from RSS feeds

### Finance
- **Crypto & Gold** - Live prices for Bitcoin, Ethereum, and Gold with 24h changes

## Adding Widgets

1. Open the Widget Gallery (+ button)
2. Browse or filter by category
3. Click a widget card to preview (optional)
4. Click "Add Widget" or "Add to Dashboard"
5. The widget appears on your dashboard immediately
6. Drag and resize to position it where you want

## Removing Widgets

To remove a widget, click the **X** button in the widget's header on your dashboard.

## Tips

- Widgets marked as "Added" are already on your dashboard
- You can add multiple instances of the same widget type
- All widgets are draggable and resizable
- Widget positions are saved automatically
- Use categories to quickly find the type of widget you need
