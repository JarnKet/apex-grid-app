# Product Overview

ApexGrid is a Chrome browser extension that replaces the default new tab page with a customizable, widget-based dashboard. It provides users with a productive workspace featuring drag-and-drop widgets including clock, calendar, todo list, quick links, daily quotes, and currency exchange rates.

## Key Features

- Drag-and-drop widget positioning with persistent layouts
- Resizable widgets with automatic collision handling
- Six core widgets: Clock, Calendar, Todo, Quick Links, Quote, Currency
- Dark/light theme support with customizable backgrounds
- Cross-device sync via Chrome Storage API
- External API integrations for quotes and currency rates

## Target Users

Users who want a productive, personalized new tab experience with quick access to essential information and tools without leaving their browser.
