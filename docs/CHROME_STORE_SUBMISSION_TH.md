# คำตอบสำหรับแบบฟอร์มส่ง Chrome Web Store

## วัตถุประสงค์เดียว (Single Purpose)
**คำอธิบาย (0-1,000 ตัวอักษร):**

ApexGrid เป็นส่วนขยายที่แทนที่หน้าแท็บใหม่ของ Chrome ด้วยแดชบอร์ดที่ปรับแต่งได้ ช่วยให้ผู้ใช้เข้าถึงข้อมูลและเครื่องมือที่จำเป็นได้อย่างรวดเร็วในที่เดียว เช่น นาฬิกา ปฏิทิน รายการสิ่งที่ต้องทำ ลิงก์ด่วน ข้อความคำคม อัตราแลกเปลี่ยน สภาพอากาศ และอื่นๆ ผู้ใช้สามารถลากวางและปรับขนาดวิดเจ็ตได้ตามต้องการ พร้อมซิงค์การตั้งค่าผ่าน Chrome Storage API

---

## เหตุผลเกี่ยวกับการขอสิทธิ์

### 1. สิทธิ์ "storage" (0-1,000 ตัวอักษร)

ส่วนขยายใช้สิทธิ์ storage เพื่อบันทึกและซิงค์การตั้งค่าของผู้ใช้ระหว่างอุปกรณ์ ได้แก่:
- ตำแหน่งและขนาดของวิดเจ็ตที่ผู้ใช้จัดวาง
- ข้อมูลในวิดเจ็ต เช่น รายการสิ่งที่ต้องทำ ลิงก์ด่วน
- การตั้งค่าธีมและพื้นหลัง
- ค่าที่ผู้ใช้กำหนดเองในแต่ละวิดเจ็ต

การใช้ Chrome Storage API ทำให้ผู้ใช้สามารถเปิดแท็บใหม่บนอุปกรณ์ใดก็ได้และเห็นแดชบอร์ดที่ตั้งค่าไว้เหมือนเดิม ซึ่งเป็นฟีเจอร์หลักของส่วนขยาย

---

### 2. สิทธิ์ "geolocation" (0-1,000 ตัวอักษร)

ส่วนขยายใช้สิทธิ์ geolocation เพื่อแสดงข้อมูลสภาพอากาศในพื้นที่ของผู้ใช้โดยอัตโนมัติผ่าน Location Widget ซึ่งเป็นหนึ่งในวิดเจ็ตที่ผู้ใช้สามารถเลือกเพิ่มได้

การใช้ geolocation ช่วยให้:
- แสดงสภาพอากาศปัจจุบันตามตำแหน่งที่ตั้งจริง
- ผู้ใช้ไม่ต้องกรอกชื่อเมืองด้วยตนเอง
- ข้อมูลสภาพอากาศแม่นยำและเกี่ยวข้องกับผู้ใช้

หมายเหตุ: ผู้ใช้สามารถปฏิเสธการให้สิทธิ์ geolocation ได้ และยังสามารถใช้งานส่วนขยายได้ปกติ โดยสามารถกรอกชื่อเมืองด้วยตนเองแทน

---

### 3. สิทธิ์ของโฮสต์ (Host Permissions) (0-1,000 ตัวอักษร)

ส่วนขยายขอสิทธิ์เข้าถึงโฮสต์ภายนอกเพื่อดึงข้อมูลสำหรับวิดเจ็ตต่างๆ:

**api.quotable.io** - ดึงข้อความคำคมสร้างแรงบันดาลใจรายวันสำหรับ Quote Widget

**api.rss2json.com** - แปลง RSS feeds เป็น JSON เพื่อแสดงข่าวและบล็อกใน RSS Feed Widget

**api.coingecko.com** - ดึงราคาและข้อมูล cryptocurrency แบบเรียลไทม์สำหรับ Crypto Widget

**api.open-meteo.com** - ดึงข้อมูลสภาพอากาศและพยากรณ์อากาศสำหรับ Weather และ Location Widget

**picsum.photos** - ดึงภาพพื้นหลังสุ่มคุณภาพสูงสำหรับตัวเลือกพื้นหลังแบบสุ่ม

API ทั้งหมดเป็นบริการสาธารณะที่ไม่ต้องการข้อมูลส่วนตัวของผู้ใช้ และใช้เพื่อเพิ่มฟังก์ชันการทำงานของวิดเจ็ตเท่านั้น

---

## หมายเหตุเพิ่มเติม

- ส่วนขยายไม่เก็บรวบรวมข้อมูลส่วนตัวของผู้ใช้
- ข้อมูลทั้งหมดถูกเก็บไว้ในเครื่องผู้ใช้และซิงค์ผ่าน Chrome Storage API เท่านั้น
- ไม่มีการส่งข้อมูลไปยังเซิร์ฟเวอร์ภายนอกของเรา
- การเรียก API ภายนอกเป็นการดึงข้อมูลสาธารณะเท่านั้น

---

## นโยบายความเป็นส่วนตัว (Privacy Policy)

**URL ของนโยบายความเป็นส่วนตัว:**

คุณต้องอัปโหลดไฟล์ `PRIVACY_POLICY.md` หรือ `PRIVACY_POLICY_TH.md` ไปยังที่ใดที่หนึ่งที่เข้าถึงได้สาธารณะ เช่น:

### ตัวเลือกที่แนะนำ:

1. **GitHub Pages** (ฟรี):
   - สร้าง repository สาธารณะบน GitHub
   - เปิดใช้งาน GitHub Pages ในการตั้งค่า repository
   - อัปโหลด PRIVACY_POLICY.md
   - URL จะเป็น: `https://[username].github.io/[repo-name]/PRIVACY_POLICY.html`

2. **GitHub Raw** (ง่ายที่สุด):
   - อัปโหลดไฟล์ไปยัง repository สาธารณะ
   - คลิกขวาที่ปุ่ม "Raw" และคัดลอก URL
   - URL จะเป็น: `https://raw.githubusercontent.com/[username]/[repo]/main/PRIVACY_POLICY.md`

3. **เว็บไซต์ของคุณเอง**:
   - อัปโหลดไปยังเว็บไซต์ส่วนตัว
   - URL: `https://yourwebsite.com/apexgrid-privacy-policy`

4. **Google Sites** (ฟรี):
   - สร้างหน้าเว็บฟรีบน Google Sites
   - วางเนื้อหานโยบายความเป็นส่วนตัว
   - เผยแพร่และใช้ URL ที่ได้

### ตัวอย่าง URL ที่ใช้ได้:
```
https://yourusername.github.io/apexgrid/privacy-policy
https://raw.githubusercontent.com/yourusername/apexgrid/main/PRIVACY_POLICY.md
https://yourwebsite.com/privacy-policy
```

**หมายเหตุ**: URL ต้องเข้าถึงได้สาธารณะและแสดงนโยบายความเป็นส่วนตัวฉบับเต็ม
