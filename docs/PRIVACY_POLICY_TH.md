# นโยบายความเป็นส่วนตัวสำหรับ ApexGrid

**อัปเดตล่าสุด:** 15 พฤศจิกายน 2568

## บทนำ

ApexGrid ("เรา" หรือ "ส่วนขยาย") มุ่งมั่นในการปกป้องความเป็นส่วนตัวของคุณ นโยบายความเป็นส่วนตัวนี้อธิบายวิธีที่ ApexGrid จัดการข้อมูลเมื่อคุณใช้ส่วนขยาย Chrome ของเรา

## ข้อมูลที่เราเก็บรวบรวม

### ข้อมูลที่เก็บในเครื่อง

ApexGrid เก็บข้อมูลต่อไปนี้ในเครื่องของคุณผ่าน Chrome Storage API:

- **การกำหนดค่าวิดเจ็ต**: ตำแหน่ง ขนาด และเลย์เอาต์ของวิดเจ็ตบนแดชบอร์ด
- **ข้อมูลวิดเจ็ต**: เนื้อหาที่ผู้ใช้ป้อน เช่น:
  - รายการสิ่งที่ต้องทำ
  - ลิงก์ด่วนและบุ๊กมาร์ก
  - บันทึกและข้อความที่กำหนดเอง
  - ค่าตัวนับและนับถอยหลัง
  - ข้อมูลการติดตามการดื่มน้ำ
  - การตั้งค่าวิดเจ็ตที่กำหนดเอง
- **การตั้งค่า**: การเลือกธีม ภาพพื้นหลัง และการตั้งค่าการแสดงผล
- **ข้อมูลแคช**: การตอบกลับ API ที่แคชชั่วคราว (คำคม สภาพอากาศ อัตราแลกเปลี่ยน)

### ข้อมูลที่เราไม่เก็บรวบรวม

- เราไม่เก็บ จัดเก็บ หรือส่งข้อมูลส่วนบุคคลใดๆ ไปยังเซิร์ฟเวอร์ของเรา
- เราไม่ติดตามประวัติการเรียกดูของคุณ
- เราไม่เก็บข้อมูลการวิเคราะห์หรือสถิติการใช้งาน
- เราไม่ขายหรือแบ่งปันข้อมูลใดๆ กับบุคคลที่สาม
- เราไม่ต้องการการสร้างบัญชีหรือการเข้าสู่ระบบ

## วิธีการใช้ข้อมูล

ข้อมูลทั้งหมดที่เก็บโดย ApexGrid ใช้เพื่อ:
- ให้ฟังก์ชันการทำงานของแดชบอร์ด
- ซิงค์การตั้งค่าของคุณระหว่างอุปกรณ์ผ่าน Chrome Sync (หากเปิดใช้งาน)
- แสดงเนื้อหาที่ปรับแต่งตามความต้องการของคุณ

## บริการของบุคคลที่สาม

ApexGrid เชื่อมต่อกับ API ภายนอกเพื่อให้ฟีเจอร์บางอย่าง บริการเหล่านี้อาจเก็บข้อมูลตามนโยบายความเป็นส่วนตัวของตนเอง:

### API ที่ใช้:

1. **Quotable.io** (https://api.quotable.io)
   - วัตถุประสงค์: ดึงข้อความคำคมสุ่ม
   - ข้อมูลที่ส่ง: ไม่มี (API สาธารณะ)

2. **RSS2JSON** (https://api.rss2json.com)
   - วัตถุประสงค์: แปลง RSS feeds เป็น JSON
   - ข้อมูลที่ส่ง: URL ของ RSS feed ที่คุณกำหนดค่า

3. **CoinGecko** (https://api.coingecko.com)
   - วัตถุประสงค์: ดึงราคา cryptocurrency
   - ข้อมูลที่ส่ง: ไม่มี (API สาธารณะ)

4. **Open-Meteo** (https://api.open-meteo.com)
   - วัตถุประสงค์: ดึงพยากรณ์อากาศ
   - ข้อมูลที่ส่ง: พิกัดทางภูมิศาสตร์ (หากอนุญาต geolocation)

5. **Picsum Photos** (https://picsum.photos)
   - วัตถุประสงค์: ให้ภาพพื้นหลังสุ่ม
   - ข้อมูลที่ส่ง: ไม่มี (API สาธารณะ)

6. **TradingView** (วิดเจ็ตฝังตัว)
   - วัตถุประสงค์: แสดงกราฟการเงินและข้อมูลตลาด
   - ข้อมูลที่ส่ง: ไม่มี (iframe ฝังตัว)

7. **Unsplash** (หากกำหนดค่า)
   - วัตถุประสงค์: ให้ภาพพื้นหลังคุณภาพสูง
   - ข้อมูลที่ส่ง: คำค้นหา (หากใช้)

## คำอธิบายสิทธิ์

### สิทธิ์ Storage
- **เหตุผลที่ต้องการ**: เพื่อบันทึกการกำหนดค่าแดชบอร์ดและซิงค์ระหว่างอุปกรณ์
- **สิ่งที่เราเข้าถึง**: เฉพาะข้อมูลที่สร้างโดย ApexGrid
- **การควบคุมของคุณ**: คุณสามารถล้างข้อมูลทั้งหมดจากการตั้งค่าส่วนขยาย

### สิทธิ์ Geolocation
- **เหตุผลที่ต้องการ**: เพื่อตรวจจับตำแหน่งของคุณโดยอัตโนมัติสำหรับวิดเจ็ตสภาพอากาศ
- **สิ่งที่เราเข้าถึง**: พิกัดทางภูมิศาสตร์โดยประมาณของคุณ
- **การควบคุมของคุณ**: คุณสามารถปฏิเสธสิทธิ์นี้ได้ วิดเจ็ตสภาพอากาศจะต้องป้อนตำแหน่งด้วยตนเอง
- **การเก็บข้อมูล**: ตำแหน่งไม่ถูกเก็บไว้ ใช้เพียงชั่วคราวเพื่อดึงข้อมูลสภาพอากาศ

### สิทธิ์โฮสต์
- **เหตุผลที่ต้องการ**: เพื่อดึงข้อมูลจาก API ภายนอกสำหรับฟังก์ชันวิดเจ็ต
- **สิ่งที่เราเข้าถึง**: เฉพาะ API endpoints ที่ระบุข้างต้น
- **การควบคุมของคุณ**: จำเป็นสำหรับวิดเจ็ตบางตัวในการทำงาน

## ความปลอดภัยของข้อมูล

- ข้อมูลทั้งหมดเก็บในเครื่องโดยใช้ Chrome Storage API ที่ปลอดภัย
- ไม่มีการส่งข้อมูลไปยังเซิร์ฟเวอร์ของเรา
- การเรียก API ไปยังบริการบุคคลที่สามใช้การเข้ารหัส HTTPS
- คุณสามารถลบข้อมูลที่เก็บไว้ทั้งหมดได้ตลอดเวลาจากการตั้งค่าส่วนขยาย

## Chrome Sync

หากคุณเปิดใช้งาน Chrome Sync ในเบราว์เซอร์:
- การตั้งค่า ApexGrid ของคุณจะซิงค์ระหว่างอุปกรณ์
- การซิงค์นี้จัดการโดย Google Chrome ทั้งหมด
- ข้อมูลถูกเข้ารหัสระหว่างการส่ง
- ดูนโยบายความเป็นส่วนตัวของ Google สำหรับ Chrome Sync

## ความเป็นส่วนตัวของเด็ก

ApexGrid ไม่เก็บข้อมูลจากเด็กอายุต่ำกว่า 13 ปีโดยเจตนา ส่วนขยายออกแบบมาสำหรับผู้ชมทั่วไปและไม่ต้องการข้อมูลส่วนบุคคลใดๆ

## การเปลี่ยนแปลงนโยบายนี้

เราอาจอัปเดตนโยบายความเป็นส่วนตัวนี้เป็นครั้งคราว การเปลี่ยนแปลงจะสะท้อนโดยการอัปเดตวันที่ "อัปเดตล่าสุด" ที่ด้านบนของนโยบาย การใช้ ApexGrid ต่อไปหลังจากการเปลี่ยนแปลงถือเป็นการยอมรับนโยบายที่อัปเดต

## การลบข้อมูล

คุณสามารถลบข้อมูลทั้งหมดที่เก็บโดย ApexGrid ได้ตลอดเวลา:
1. เปิดการตั้งค่า ApexGrid
2. คลิก "รีเซ็ตการตั้งค่าทั้งหมด" หรือ "ล้างข้อมูลทั้งหมด"
3. หรือถอนการติดตั้งส่วนขยายเพื่อลบข้อมูลที่เก็บไว้ทั้งหมด
